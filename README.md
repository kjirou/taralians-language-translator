# taralians-language-translator

A fictional "The Taralians Language" translator


## これは何？

初代ロマサガに登場する架空の言語である「タラール語」の翻訳機です。


## インストール

```
npm install -g taralians-language-translator
```


## 使い方

```
taralians "What is your name?"
ムアナ ウト ホオテ ラヘ?
```
